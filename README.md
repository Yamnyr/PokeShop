# PokeShop

## Project Overview
Un projet de catalogue ce darte pokemon développé par Marine, Corentin, Quentin composé de deux sous-projets principaux : **POKEFRONT** (React) et **POKEBACK** (Node Express).

## Sous-Projets

### POKEFRONT (Frontend)
- **Technologie** : React
- **Configuration** :
```bash
cd pokefront
npm install
npm start
```

### POKEBACK (Backend)
- **Technology**: Node.js with Express / mongodb
- **Setup**:
```bash
cd pokeback
npm install
npm start
```
