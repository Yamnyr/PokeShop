# PokeShop

## Project Overview
Un projet développé par Quentin Waroquet, composé de deux sous-projets principaux : **LCSFRONT** (React) et **LCSBACK** (Node Express).

## Sous-Projets

### LCSFRONT (Frontend)
- **Technologie** : React
- **Configuration** :
```bash
cd pokefront
npm install
npm start
```

### LCSBACK (Backend)
- **Technology**: Node.js with Express / mongodb
- **Setup**:
```bash
cd pokeback
npm install
npm start
```
